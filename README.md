# Projeto-Integrado-I

Este projeto integrado tem como objetivo apresentar segurança digital voltada para pequenas empresas, com foco em orientar sobre boas práticas de proteção de dados e segurança da informação.

## Participantes
Ana Julia Evangelista da Silva
Guilherme Lo Castro Mendes
Matheus Santos Oliveira
Vinicius Alves de Oliveira Finati

## Objetivo
Identificar ameaças comuns à segurança digital em pequenos negócios, Apresentar práticas simples e de baixo custo para proteger dados, Orientar funcionários sobre boas práticas no uso de sistemas e internet, Introduzir conceitos básicos da LGPD e privacidade de dados.
